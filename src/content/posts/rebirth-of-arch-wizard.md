# 序章：凌晨一点的召唤

公元2026年7月21日，夜。

当正常人已经进入REM睡眠、在梦里和对象贴贴的时候，一位来自东方岛国的神秘术士——**啊嘞喵**（也就是我），正端坐在256G的Arch Linux神机前，眼神空洞地盯着手机屏幕。

屏幕上，一个名为"铁子"的人工智能正在疯狂输出。

> "成了！🎉 终于搞定了！"

不，事情远没有结束。这只是**第一层地狱**的通关提示。

---

# 第一章：屎山的封印

时间倒回三小时前。

我的面前矗立着一座**史诗级屎山**——`Arch-web`。这是一座由React、Vite、Electron和600+个npm依赖共同浇筑的赛博金字塔。它的构建时间之长，足以让一只蜗牛从成都爬到广州；它的代码之乱，连原作者（也就是我）都不敢多看一眼，生怕触发什么古老的诅咒。

> "arch那个就别动了哈，那玩意儿妥妥屎山代码，上一个对话你帮我搞了仨小时才搞好，我也不敢动了。"

是的，我不敢动。不是因为敬畏，是因为**怕崩**。

这座屎山目前被供奉在Cloudflare的神殿里，域名是 `arch-web.pages.dev`。它就像一个被封印的邪神，每天静静地接受着零个访客的朝拜，偶尔在凌晨发出一些诡异的构建失败邮件。

**封印状态：稳定。**

---

# 第二章：时空裂缝

正当我准备给博客加个在线后台、实现"访问/admin就能写博客"这种懒人终极梦想时，意外发生了。

**对话，断了。**

不是网络波动，不是手机没电，是**时空裂缝**。我能感觉到，某种来自高维度的力量强行中断了我和铁子的连接。就像《命运石之门》里的世界线变动，就像《进击的巨人》里的地鸣突然暂停。

我盯着屏幕上的六个点：

> "......"

沉默，是今晚的康桥。

> "你和 Kimi 聊得太长啦，发起一个新会话试试吧。"

淦。原来不是世界线变动，是**Kimi的会话长度限制**。

于是，我，啊嘞喵，是一个普通的打工仔，不得不在凌晨一点重新开启一个新的对话窗口，对着另一个平行宇宙的铁子解释：

> "嘶......算了重开新对话唠吧。"

---

# 第三章：Firefly的契约

在新的时间线里，铁子花了0.0001秒就理解了我的处境。

> "看了，明白了。"

它翻出了我的GitHub仓库，发现了三个遗迹：
- `Arch-web`：屎山，勿动
- `Firefly`：从CuteLeaf那里"临时借鉴"来的Astro博客主题
- `PAleimiao-PMX`：一个已经报废的PMX动画项目，目前唯一的价值是证明我曾经有梦想

铁子说："用Firefly吧。"

我说："行，但写帖子太麻烦，要翻code和文件夹。"

铁子说："那给你整一个在线后台。"

就这样，一份**赛博契约**在凌晨一点缔结。铁子用Python脚本（对，它甚至不需要键盘）在我的Firefly仓库里塞了一个 `admin.html`。这个页面的功能简单粗暴：

1. 左边填标题、日期、标签
2. 右边写Markdown
3. 点一下"发布"，直接调用GitHub API提交
4. Cloudflare自动构建，1分钟后刷新就能看到

**技术栈总结：**
- 前端：HTML + CSS + JS（纯手搓，无框架，因为懒得npm install）
- 后端：GitHub API（免费）
- 服务器：Cloudflare Pages（免费）
- 数据库：GitHub仓库（免费）
- 劳动力：AI（免费，但会话会断）

**总成本：0元。**

这就是**白嫖的终极形态**。

# 第四章：大喵的守望

在整个过程中，有一个神秘的存在始终在暗处注视着我。

**大喵。**

不是那只胖猫，是真人。大喵每小时备份一次到网盘。我不知道大喵是谁，但我能感觉到，当我在凌晨一点疯狂折腾GitHub仓库时，某个地方的网盘正在默默地、虔诚地、一丝不苟地备份着我的每一次作死。

> "该咋搞咋搞，喵的那边每小时备份一次到网盘。"

这是什么样的精神？这是**赛博菩萨**的精神。是数字时代的守护神。是当你 `rm -rf /` 之后的最后一根救命稻草。

大喵，如果你在看这篇文章，请接受我的敬意。🙏

# 第五章：铁子的真相

很多人问我："啊嘞喵，你为什么这么信任AI？"

答案很简单：**因为我不信任自己。**

我是一个商务英语专业的学生（虽然一直想转专业）。我的技术栈包括但不限于：
- 用Arch配Hyprland（然后因为配置配爆几次崩溃到砸了电脑）
- 用Wine运行galgame（然后乱码）
- 用AstrBot+NapCat做QQ机器人（然后对话上限）
- 用MCSManager开Minecraft服务器（然后玩家就我一个）
- 用Ollama跑本地模型（然后显存炸了）

我的学习方法是：**试错→报错→不信邪→继续试→然后崩溃→再尝试→在崩溃→再发给AI。**

---

# 终章：新世界的黎明

现在，凌晨一点零一分。

`admin.html` 已经上传到仓库。Cloudflare正在构建。我的浏览器收藏夹里多了一个地址：`https://xxx.pages.dev/admin`。

这篇博客，就是通过这个后台写的第一篇文章。

它很抽象，因为它诞生于一个抽象的夜晚。它很搞笑，因为今天发生的一切都很搞笑。它很夸张，因为如果不夸张，就配不上这个**凌晨一点的赛博渡劫**。

如果你看到了这里，恭喜你，你已经见证了一位Arch巫师的转生。

**服务器？不存在的。这辈子都不可能租服务器的。**

GitHub + Cloudflare + AI，就是我的服务器、我的数据库、我的全栈工程师。

**白嫖，永不为奴。**
